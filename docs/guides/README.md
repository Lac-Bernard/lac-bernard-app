# Member area guides

Plain-language help for the **member area** and **membership administration** on the Lac Bernard Association website. These files are the **source of truth** in the repository. You can copy sections into Google Docs or export to PDF for distribution.

| File | Language | Audience |
|------|----------|----------|
| [member-area-member-en.md](member-area-member-en.md) | English | Members and households |
| [member-area-member-fr.md](member-area-member-fr.md) | French | Membres et foyers |
| [member-area-admin-en.md](member-area-admin-en.md) | English | Volunteers with admin access |
| [member-area-admin-fr.md](member-area-admin-fr.md) | French | Bénévoles avec accès admin |

**Not covered here:** editing news and general website pages in **TinaCMS** (separate tool at `/admin` on the site).

When updating guides after UI changes, align wording with [`src/lib/members/i18n.ts`](../../src/lib/members/i18n.ts).
